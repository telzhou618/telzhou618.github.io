# Mac 更换控制台 shell


## shell 常用命令
```shell
more /etc/shells 	# 查查有多少个shell
echo $SHELL 		# 查看当前shell
chsh -s /bin/bash   # 切换到 bash
chsh -s /bin/zsh	# 切换到zsh
chsh -s /usr/local/bin/fish	#切换到fish
```
